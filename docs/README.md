# Dokumentasi Rizqhub

Dokumentasi ini menjadi sumber keputusan produk, arsitektur, implementasi, dan operasi Rizqhub.

## Fondasi platform

- [`Rizqhub_Platform_Foundation_v1.1.md`](Rizqhub_Platform_Foundation_v1.1.md): Product Vision, Product Principles, Platform Blueprint, Domain Model, strategi migrasi, dan roadmap.
- [`adr/`](adr/): Architecture Decision Records yang menjelaskan keputusan, alternatif, alasan, dan konsekuensi.
- [`M0_M1_Implementation_Spec_v1.0.md`](M0_M1_Implementation_Spec_v1.0.md): ruang lingkup, backlog, gate, dan exit criteria M0-M1.
- [`BASELINE_PROVENANCE.md`](BASELINE_PROVENANCE.md): identitas dan keterbatasan bukti baseline source v1.0.1.

## Kontrol implementasi M0

- Status bukti: [`implementation/M0_STATUS.md`](implementation/M0_STATUS.md)
- Prosedur operasional: [`operations/`](operations/)
- Manifest baseline: [`baseline-files.sha256`](baseline-files.sha256)

Perubahan arsitektur utama wajib dicatat sebagai ADR. Perubahan database dan rilis harus mengikuti checklist serta runbook di folder `operations`.
